## Logical AND

### `&&` operator
    
    !js
    var v = v && 15;
    console.log(v);

Experiment by first setting `v` to `100`, `0`, `null`.

### Exmaple:

    !js
    var test = true;
    test && console.log('test is true!');

## Logical OR

### `||` operator
    
    !js
    var v = v || 10;

Experiment by first setting `v` to `100`, `0`, `null`.

### Exmaple:

    !js
    var empty = "";
    var myString = empty || "Hello Web :)";
    console.log(myString);
